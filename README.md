# On-screen Keyboard
On-screen keyboard app for SVH

## Install dependencies

``` Yarn ```

## Firebase
Don't forget to include ```google-services.json``` file

## Link packages
Make sure packages are linked with ``` react-native link ```

## Run project
Android: ``` react-native run-android ```
